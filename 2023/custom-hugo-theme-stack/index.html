新增菜单选项

有两种办法：

  1. 按照文档 Custom Menu 在 config.yaml 添加新菜单选项，并在 Tabler Icons 下载 icon
  2. 找到主题自带菜单选项（Home、About、Archive…）所在目录，照猫画虎，举例：在 /content/page 目录下新建<选项名>文件夹，照着默认文件创建 index.md

禁止部分页面显示 ReadingTime

比如 /friends ，不是文章，不需要显示阅读时间

1
2
3

# /blog/layouts/partials/article/components/details.html

{{ if and (not .Date.IsZero) (.Site.Params.article.readingTime) }}

默认逻辑是 if or ，改成 if and 之后只有 Front Matter 里 date 参数为 true 的页面会显示阅读时长
